# BLOGLOADER

Code for the bootloader tutorial series I'm writing at: https://kevintrieu.com/writing/bootloader/intro/

You'll need to have Rust/Cargo and QEMU installed to run. You'll also need [OVMF](https://github.com/tianocore/tianocore.github.io/wiki/OVMF) to boot QEMU - for convenience I've included a version. You probably don't need to update it, but in case you're curious the bundled version is `2022.02-3ubuntu0.22.04.5`
